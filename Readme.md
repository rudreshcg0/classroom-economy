# Virtual Classroom Economy System
A multi-school virtual economy for teachers to reward students.

## Setup Instructions
1. **Database**: 
   - Install PostgreSQL & pgAdmin4.
   - Run the script found in `/sql/database_schema.sql`.
2. **Java**:
   - Add the JDBC Driver in `lib/` to your project's referenced libraries.
3. **Server**:
   - Run using Apache Tomcat 10.
